# Custom Test Tool

Windows 창의 투명도와 작업 표시줄 표시 여부를 제어하는 도구입니다.

## 기능

- ✅ 창 투명도 조절 (0-100%)
- ✅ 작업 표시줄 숨김/표시
- ✅ 창 검색 필터
- ✅ 실시간 상태 표시
- ✅ 트레이 아이콘으로 숨김
- ✅ 창마다 개별 설정 저장

## 다운로드

[Releases 페이지](https://github.com/dolbu95/Chrome-Tool/releases)에서 최신 버전의 `CustomTestTool.exe`를 다운로드하세요.

## 사용 방법

1. `CustomTestTool.exe` 실행
2. 필터에 창 이름 입력 (예: "Chrome", "Purple")
3. "Refresh List" 클릭
4. 목록에서 창 선택
5. 투명도 슬라이더 조절 또는 작업 표시줄 체크박스 변경

## 종료 방법

- **최소화**: 창 최소화 버튼(-) 또는 X 버튼 클릭 → 트레이로 숨김
- **복원**: 트레이 아이콘 더블클릭
- **완전 종료**: 트레이 아이콘 우클릭 → Quit 선택
